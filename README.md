# parse-jbse-output

Python scripts to parse a JBSE output.


```sh
$ python src/main.py -a parse -p examples/1 -m 0

[Path: path3.txt]
((0<=s.value.length)&&(!(s.value.length<=0))&&(!(s.value[0]==(char)40)))
[Path: path4.txt]
```
